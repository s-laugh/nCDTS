Fetched nCDTS Instructions Example

Additional Script Instructions, Component Replacement Instructions, Direct Replacement Instructions

This page demonstrates how to provide CDTS instructions from a JSON end point. The script element that contains the CDTS instruction object should be place at the end of the body element before the script element for the 'cdtsnext.js'. The CDTS instruction object variable name must be cdts_instructions_url.

Example:

                <body>
                ...
                <script>
                var cdts_instructions_url = "/cdts-instructions.json"
                </script>
                <script src="/cdtsnext.js"></script>
            </body>

The following examples are using the CDTS instructions from this JSON file.

Additional Script Instructions

The following 'w3-card' div element content is being replaced by java script that is being loaded into the page using the addtional script instructions:

        "additional-scripts": [
            "/js/demo1.js",
            "/js/demo2.js"
        ]

The demo1.js script should replace this content.

The demo2.js script should replace this content.

Component Replacement Instructions

The following 'w3-card' div element content is being replaced using the Demonstration Components module:

            <script src="/democomponents.js"></script>
            <script src="/cdtsnext.js"></script>
            </body>

The component replacement instructions:

        "component-replacements": {
            "div-id-three": {
                "component-package": "democomponents",
                "component-name": "LoremIpsumComponent",
                "output": "p",
                "number": 2
            },
            "div-id-four": {
                "component-package": "democomponents",
                "component-name": "LoremIpsumComponent",
                "output": "ul",
                "number": 3
            },
            "div-id-five": {
                "component-package": "democomponents",
                "component-name": "HelloWorldComponent",
                "name": "nCDTS",
            }
        }
