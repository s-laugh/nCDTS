Inline nCDTS Instructions Example

Additional Script Instructions

This page demonstrates how to provide CDTS instructions inline (in a script tag) on a page. The script element that contains the CDTS instruction object should be place at the end of the body element before the script element for the 'cdtsnext.js'. The CDTS instruction object variable name must be cdts_instructions.

Example:

                <body>
                ...
                <script>
                    var cdts_instructions = {
                    ...
                    };
                </script>
                <script src="/cdtsnext.js"></script>
            </body>

Additional Script Instructions

The following cards are an example of an additional script instruction to the CDTS. A additional script instruction loads a script from the specified source by creating a new script element at the end of the body of the page.

The JSON format for additional script instructions is: 

            "additional-scripts": [
            "/script-1.js",
            "/script-2.js",
            ...
        ]

The following 'w3-card' div element content is being replaced by java script that is being loaded into the page using the addtional script instructions:

        "additional-scripts": [
            "/js/demo1.js",
            "/js/demo2.js"
        ]

The demo1.js script should replace this content.

The demo2.js script should replace this content.
