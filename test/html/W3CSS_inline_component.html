Inline nCDTS Instructions Example

Component Replacement Instructions

This page demonstrates how to provide CDTS instructions inline (in a script tag) on a page. The script element that contains the CDTS instruction object should be place at the end of the body element before the script element for the 'cdtsnext.js'. The CDTS instruction object variable name must be cdts_instructions.

Example:

                <body>
                ...
                <script>
                    var cdts_instructions = {
                    ...
                    };
                </script>
                <script src="/cdtsnext.js"></script>
            </body>

Component Replacement Instructions

The following cards are an example of a component replacement instruction to the CDTS. A component replacement finds an element specified by an id attribute and replaces the inner HTML of that element with HTML from the components generateHtml() function.

The JSON format for component replacement instructions is: 

            "component-replacements":
            {
                "element-id": {
                    "component-package": "componentspackage",
                    "component-name": "componentname",
                    ... addtional data that will be passed to component ...
                },
                "element-id": {
                    "component-package": "componentspackage",
                    "component-name": "componentname",
                    ... addtional data that will be passed to component ...
                },
                ...
            }

Components are not part of the core nCDTS module. Currently component modules must be added using a script tag at the end of the body element before the nCDTS script element. A future goal is to have nCDTS module load component modules without the need for an additional script element.

The following 'w3-card' div element content is being replaced using the Demonstration Components module:

            <script src="/democomponents.js"></script>
            <script src="/cdtsnext.js"></script>
            </body>

The component replacement instructions:

        "component-replacements": {
            //Lorem Ipsum text - 2 headers and paragraphs
            "div-id-one": {
                "component-package": "democomponents",
                "component-name": "LoremIpsumComponent",
                "output": "p",
                "number": 2
            },
            //Lorem Ipsum un-ordered list , 3 items
            "div-id-two": {
                "component-package": "democomponents",
                "component-name": "LoremIpsumComponent",
                "output": "ul",
                "number": 3
            },
            // Hello nCDTS
            "div-id-three": {
                "component-package": "democomponents",
                "component-name": "HelloWorldComponent",
                "name": "nCDTS",
            }
        }
